# 有限元基础

有限元法(FEM)是一种数值分析方法，用于求解复杂工程问题。

## 基本原理

### 1. 离散化

将连续体离散为有限个单元的组合：
- 1D: 杆单元、梁单元
- 2D: 平面应力/应变单元、三角形、四边形
- 3D: 四面体、六面体单元

### 2. 单元插值

```
u(x) = Σ Ni(x) * ui
```

其中 Ni 为形函数，ui 为节点位移。

### 3. 刚度矩阵

```
[K]{u} = {F}
```

## 常见单元类型

### 1. 杆单元 (Truss)
- 2节点，2自由度/节点
- 只能承受轴向力

### 2. 梁单元 (Beam)
- 2节点，6自由度/节点
- 能承受弯矩和剪力

### 3. 平面单元
- 三角形单元 (T3, T6)
- 四边形单元 (Q4, Q8)

### 4. 实体单元
- 四面体单元 (Tet4, Tet10)
- 六面体单元 (Hex8, Hex20)

## 网格质量
