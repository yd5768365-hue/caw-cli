# SolidWorks学习笔记

> 整理者：双非大学生
> 适用：机械专业学生 / CAD初学者

## 📌 学习目标

- 掌握SolidWorks基本操作
- 能够进行三维建模
- 理解装配体设计流程
- 学会创建工程图

---

## 第一章 入门基础

### 1.1 界面介绍

**主要区域**：
- 菜单栏：文件、编辑、视图、插入、工具
- 工具栏：常用命令快捷按钮
- 特征管理器：显示模型结构树
- 属性管理器：设置特征参数
- 图形区域：显示3D模型

### 1.2 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 保存 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Ctrl+8 | 等轴测视图 |
| Ctrl+7 | 前视图 |
| Space | 视图定向 |

## 第二章 草图绘制

### 2.1 草图基本命令

| 命令 | 用途 | 快捷键 |
|------|------|--------|
| 直线 | 绘制直线 | S → L |
| 圆 | 绘制圆 | S → C |
| 圆弧 | 绘制圆弧 | S → A |
| 矩形 | 绘制矩形 | S → R |
| 多边形 | 绘制多边形 | - |
| 剪裁 | 剪裁草图 | S → X |
| 智能尺寸 | 添加尺寸 | S → D |

### 2.2 几何关系

**自动几何关系**：
- 相切
- 水平
- 垂直
- 共线
- 同心

**手动添加几何关系**：
- 选中元素 → 点击属性管理器中的几何关系

### 2.3 草图绘制流程

```
新建零件 → 进入草图 → 绘制轮廓 → 添加几何关系 → 标注尺寸 → 退出草图
```

## 第三章 基础特征

### 3.1 拉伸特征

**创建步骤**：
1. 绘制草图
2. 点击拉伸凸台
3. 设置深度
4. 确定

**重要参数**：
- 深度：从草图平面开始拉伸的距离
- 拔模角度：带斜度的拉伸
- 薄壁特征：创建薄壁零件

### 3.2 旋转特征

**创建步骤**：
1. 绘制旋转轮廓和中心线
2. 点击旋转凸台/基体
3. 设置旋转角度（默认360°）
4. 确定

### 3.3 扫描特征

**创建步骤**：
1. 创建扫描路径
2. 创建扫描轮廓（垂直于路径）
3. 点击扫描
4. 确定

### 3.4 放样特征

**创建步骤**：
1. 创建多个截面草图
2. 绘制引导线（可选）
3. 点击放样凸台/基体
4. 选择截面
5. 确定

## 第四章 工程特征

### 4.1 倒角

**类型**：
- 角度距离：给定角度和距离
- 距离距离：给定两个距离
- 顶点：只在一个顶点倒角

### 4.2 圆角

**类型**：
- 恒定半径：固定半径圆角
- 变半径：半径渐变
- 完整圆角：三个面组圆角

### 4.3 抽壳

**创建步骤**：
1. 选择要移除的面
2. 点击抽壳
3. 设置壁厚
4. 确定

### 4.4 孔

**类型**：
- 简单孔：普通钻孔
- 异型孔：标准孔（螺纹孔、锥孔等）

---

## 第五章 装配体

### 5.1 配合关系

| 配合类型 | 作用 |
|----------|------|
| 重合 | 两个面/线/点完全重合 |
| 平行 | 两个面/线保持平行 |
| 垂直 | 两个面/线保持垂直 |
| 同心 | 两个圆/孔中心对齐 |
| 距离 | 给定两个面/线的距离 |
| 角度 | 给定两个面的夹角 |

### 5.2 装配体设计流程

```
新建装配体 → 插入第一个零件(固定) → 插入其他零件 → 添加配合关系 → 检查干涉 → 保存
```

### 5.3 爆炸视图

**创建步骤**：
1. 点击爆炸视图
2. 选择零件 → 拖动到爆炸位置
3. 设置步骤距离
4. 确定

## 第六章 工程图

### 6.1 创建工程图

**步骤**：
1. 新建工程图
2. 选择模板（A4、A3等）
3. 添加视图：标准三视图、模型视图等
4. 添加尺寸和注解

### 6.2 视图类型

| 视图类型 | 说明 |
|----------|------|
| 标准三视图 | 主视图、俯视图、左视图 |
| 等轴测视图 | 三维立体图 |
| 剖视图 | 内部结构 |
| 局部视图 | 局部放大 |
| 断开的剖视图 | 部分剖切 |

### 6.3 尺寸标注

**类型**：
- 智能尺寸：自动检测边线
- 水平/竖直尺寸：特定方向
- 角度尺寸：角度标注
- 直径/半径：圆孔、圆弧

## 💡 建模技巧

### 1. 优先建立基准
- 基准面：三个默认基准面
- 基准轴：重要几何轴线
- 基准点：关键位置点

### 2. 善用父子关系
- 草图是特征的基础
- 修改草图，特征自动更新

### 3. 避免复杂操作
- 先简化，再细节
- 步步为营，及时保存

### 4. 常用检查
- 重建模型（Ctrl+B）
- 质量特性（检查体积、重心）
- 干涉检查（装配体）

---

## ⚠️ 常见问题

### 1. 无法拉伸
- 检查草图是否封闭
- 检查是否有过定义

### 2. 特征重建失败
- 查看失败特征
- 检查尺寸和几何关系

### 3. 装配体松动
- 检查配合关系
- 添加更多配合

### 4. 工程图比例不对
- 调整图纸比例
- 检查模型单位
