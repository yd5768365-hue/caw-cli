# 紧固件

## 概述

紧固件是用于连接两个或多个零件的机械元件，包括螺栓、螺母、垫圈、销钉等。正确选择紧固件对于确保机械连接的可靠性和安全性至关重要。

## 紧固件选型指南

### 选择原则

1. **强度匹配**：紧固件强度应与连接件强度相匹配
2. **防松措施**：根据振动情况选择合适的防松方式
3. **腐蚀环境**：选择合适材料（不锈钢、镀锌等）
4. **安装空间**：考虑扳手空间和操作便利性
5. **成本考虑**：在满足要求的前提下选择经济型

### 常见失效模式

| 失效模式 | 原因 | 预防措施 |
|----------|------|----------|
| 拉伸断裂 | 预紧力过大或工作载荷过大 | 控制预紧力，合理选型 |
| 剪切破坏 | 横向载荷过大 | 增加抗剪元件（销、键） |
| 疲劳断裂 | 交变载荷作用 | 降低应力集中，提高表面质量 |
| 松动失效 | 振动导致预紧力下降 | 采用锁紧装置，定期检查 |
| 腐蚀破坏 | 环境腐蚀作用 | 选择耐腐蚀材料，表面处理 |
