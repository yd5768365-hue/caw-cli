# 公差配合

## 概述

公差配合是机械设计中的重要概念，它决定了零件之间的装配关系和使用性能。合理的公差设计可以保证产品的功能、可靠性和经济性。

## 公差基本概念

### 尺寸公差术语

| 术语 | 符号 | 定义 | 说明 |
|------|------|------|------|
| 基本尺寸 | D | 设计给定的尺寸 | 理论尺寸，通常为整数 |
| 实际尺寸 | Da | 通过测量得到的尺寸 | 实际制造出的尺寸 |
| 极限尺寸 | Dmax, Dmin | 允许的最大和最小尺寸 | 实际尺寸应在此范围内 |
| 上偏差 | ES, es | 最大极限尺寸减基本尺寸 | 孔用ES，轴用es |
| 下偏差 | EI, ei | 最小极限尺寸减基本尺寸 | 孔用EI，轴用ei |
| 公差 | TD, Td | 允许尺寸的变动量 | TD = ES - EI, Td = es - ei |

### 公差带示意图

```
         零线
         ↑
最大极限尺寸 ──────┬───── ES/es
                  │
基本尺寸 ────────┼───── 基本尺寸线
                  │
最小极限尺寸 ──────┴───── EI/ei
```

### 公差等级选择指南

| 应用场景 | 推荐公差等级 | 理由 |
|----------|--------------|------|
| 量块、标准量具 | IT01-IT1 | 需要极高的精度和稳定性 |
| 精密轴承配合 | IT5-IT6 | 保证旋转精度和使用寿命 |
| 机床主轴 | IT5-IT7 | 平衡精度和加工成本 |
| 齿轮、联轴器 | IT7-IT8 | 满足传动精度要求 |
| 一般机械零件 | IT8-IT10 | 经济性好，满足大多数需求 |
| 冲压件、焊接件 | IT10-IT12 | 考虑工艺特点和经济性 |
| 铸件、锻件 | IT11-IT13 | 材料成型精度限制 |

## 配合类型 (Fit Types)

### 间隙配合 (Clearance Fits)

| 配合代号 | 特点 | 最小间隙 | 最大间隙 | 应用示例 |
|----------|------|----------|----------|----------|
| **H7/g6** | 极小间隙 | 接近零 | 较小 | 精密滑动，如分度头主轴 |
| **H7/f7** | 小间隙 | 较小 | 中等 | 精密转动，如机床主轴 |
| **H8/f7** | 中等间隙 | 中等 | 较大 | 一般转动，如齿轮与轴 |
| **H9/d9** | 大间隙 | 较大 | 大 | 低速转动，如皮带轮 |
| **H11/c11** | 很大间隙 | 很大 | 很大 | 粗糙转动，如农业机械 |

| 配合代号 | 特点 | 可能间隙 | 可能过盈 | 应用示例 |
|----------|------|----------|----------|----------|

| **H7/k6** | 小过盈过渡 | 很小 | 小 | 精确定位，如齿轮与轴 |
| **H7/n6** | 过盈过渡 | 无或很小 | 中等 | 较重定位，传递小扭矩 |

|----------|------|----------|----------|----------|----------|
| **H7/p6** | 小过盈 | 较小 | 中等 | 压力装配 | 定位销、定位套 |
| **H7/s6** | 中等过盈 | 中等 | 较大 | 压力或热装 | 齿轮、联轴器 |

## 基准制 (Datum Systems)

### 基孔制 (Hole Basis System)

- **特点**：孔的公差带固定，通过改变轴的公差带来获得不同配合
- **优点**：加工孔比加工轴困难，固定孔公差可减少刀具、量具数量
- **应用**：大多数机械制造中采用基孔制

### 基轴制 (Shaft Basis System)

- **特点**：轴的公差带固定，通过改变孔的公差带来获得不同配合
- **优点**：当使用冷拉轴（标准轴）时经济性好
- **应用**：农业机械、纺织机械、使用标准轴的场合

## 几何公差 (Geometric Tolerancing)

### 形状公差 (Form Tolerances)

| 公差类型 | 符号 | 定义 | 应用 |
|----------|------|------|------|
| 直线度 | — | 实际直线对理想直线的允许变动量 | 导轨、轴心线 |
| 平面度 | ⏥ | 实际平面对理想平面的允许变动量 | 安装面、基准面 |
| 圆度 | ○ | 实际圆对理想圆的允许变动量 | 轴、孔的横截面 |
| 圆柱度 | /○/ | 实际圆柱面对理想圆柱面的允许变动量 | 精密轴、孔 |

### 位置公差 (Position Tolerances)

| 公差类型 | 符号 | 定义 | 应用 |
|----------|------|------|------|
| 平行度 | // | 实际要素对基准在平行方向上的允许变动量 | 导轨、平板 |
| 垂直度 | ⟂ | 实际要素对基准在垂直方向上的允许变动量 | 立板、端面 |
| 倾斜度 | ∠ | 实际要素对基准在指定角度方向上的允许变动量 | 斜面、锥面 |
| 同轴度 | ◎ | 实际轴线对基准轴线的允许变动量 | 多段轴、套筒 |
| 对称度 | ⇔ | 实际中心要素对基准中心要素的允许变动量 | 键槽、花键 |
| 位置度 | ⌖ | 实际要素对理想位置的允许变动量 | 孔组、螺栓孔 |

### 跳动公差 (Runout Tolerances)

| 公差类型 | 符号 | 定义 | 应用 |
|----------|------|------|------|
| 圆跳动 | ↗ | 实际要素绕基准轴线旋转一周时的允许变动量 | 旋转零件 |
| 全跳动 | ↻ | 实际要素绕基准轴线连续旋转时的允许变动量 | 精密旋转件 |

## 表面粗糙度 (Surface Roughness)

### 粗糙度等级

| 等级代号 | Ra (μm) | Rz (μm) | 加工方法 | 应用 |
|----------|---------|---------|----------|------|
| N1 | 0.012 | 0.05 | 研磨、超精加工 | 量块、精密量仪 |
| N2 | 0.025 | 0.1 | 研磨、精密磨削 | 精密轴承、高精度导轨 |
| N3 | 0.05 | 0.2 | 精密磨削 | 机床主轴、精密配合面 |
| N4 | 0.1 | 0.4 | 磨削、精车 | 一般轴承、重要配合面 |
| N5 | 0.2 | 0.8 | 精车、精铣 | 齿轮工作面、导向面 |
| N6 | 0.4 | 1.6 | 车、铣、铰 | 一般配合面、密封面 |
| N7 | 0.8 | 3.2 | 车、铣、钻 | 非配合面、安装面 |
| N8 | 1.6 | 6.3 | 粗车、粗铣 | 非接触面、铸件表面 |
| N9 | 3.2 | 12.5 | 粗加工、铸造 | 毛坯面、非加工面 |
| N10 | 6.3 | 25 | 粗加工、锻造 | 粗加工面 |
| N11 | 12.5 | 50 | 不加工 | 原材料表面 |
| N12 | 25 | 100 | 不加工 | 原材料表面 |

### 粗糙度选择原则

1. **功能要求**：配合面要求高，非配合面要求低
2. **运动情况**：高速运动面要求高，静止面要求低
3. **密封要求**：密封面要求高，非密封面要求低
4. **美观要求**：外观面要求高，内部面要求低
5. **加工成本**：粗糙度每提高一级，加工成本显著增加

## 公差配合标注示例

### 尺寸公差标注

```
Ø50H7     → 基孔制，孔公差H7
Ø50f6     → 基轴制，轴公差f6
Ø50H7/f6  → 基孔制间隙配合
```

### 几何公差标注

```
[⏥|0.02]    → 平面度公差0.02mm
[//|0.01|A] → 相对于基准A的平行度0.01mm
[◎|Φ0.01|A-B] → 相对于基准A-B的同轴度Φ0.01mm
```

### 完整标注示例

```
轴的标注：
Ø30±0.01           → 直径30mm，对称公差±0.01mm
Ø30g6              → 直径30mm，公差等级g6
Ø30g6([○|0.005])   → 直径30mm，g6公差，圆度0.005mm
```

## 公差设计原则

### 经济性原则

1. **尽量选用标准公差**：IT6-IT11是经济精度范围
2. **尽量采用基孔制**：减少刀具、量具种类
3. **合理选择配合**：在满足功能的前提下选择最宽松的配合
4. **考虑工艺能力**：与企业的加工能力相匹配

### 功能原则

1. **明确功能要求**：根据零件的功能确定公差
2. **考虑装配关系**：公差链分析，避免累积误差过大
3. **考虑使用环境**：温度、湿度、振动等对公差的影响
4. **考虑磨损寿命**：运动副需要预留磨损余量

### 工艺原则

1. **考虑加工方法**：不同加工方法能达到的精度不同
2. **考虑测量方法**：公差应能被现有测量手段检测
3. **考虑装配方法**：过盈配合需要考虑装配工艺
4. **考虑热处理变形**：热处理后需要加工的要留余量

## 总结

公差配合设计是机械设计中的关键技术，需要综合考虑功能、工艺、经济等多方面因素。良好的公差设计可以：
1. 保证产品的性能和质量
2. 降低制造成本
3. 提高装配效率
4. 延长产品寿命

在实际设计中，建议遵循"够用就好"的原则，避免不必要的精度要求，同时要充分考虑工艺实现的可行性。
