# 材料力学

材料力学是研究材料在受力时变形和破坏规律的学科。

## 基础概念

### 1. 应力 (Stress)

应力是单位面积上的内力：
- 正应力 σ = F/A
- 剪应力 τ = F/A

### 2. 应变 (Strain)

应变是单位长度的变形量：
- 线应变 ε = ΔL/L

### 3. 弹性模量 (Young's Modulus)

E = σ/ε，反映材料的刚度

## 基本变形

### 1. 轴向拉伸/压缩

```
σ = F/A
ε = ΔL/L
ΔL = FL/(AE)
```

### 2. 扭转

```
τ = T*r/J
θ = TL/(GJ)
```

### 3. 弯曲

```
σ = My/I
```

