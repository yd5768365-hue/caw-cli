# 常用材料

## 概述

材料的选择直接影响机械产品的性能、寿命和成本。了解各种材料的力学性能、加工性能和应用场景是机械设计的基础。

## 材料选择原则

### 选型考虑因素

1. **力学性能**：强度、硬度、韧性、疲劳强度
2. **物理性能**：密度、导热性、导电性、热膨胀系数
3. **化学性能**：耐腐蚀性、抗氧化性
4. **加工性能**：铸造性、锻造性、焊接性、切削性
5. **经济性**：材料成本、加工成本、维护成本
6. **可用性**：市场供应、标准规格

### 典型应用建议

| 零件类型 | 推荐材料 | 理由 |
|----------|----------|------|
| 一般结构件 | Q235, Q345 | 成本低，焊接性好 |
| 重要轴类 | 45钢, 40Cr | 综合机械性能好 |
| 齿轮 | 20CrMnTi, 40Cr | 表面硬度高，心部韧性好 |
| 轴承 | GCr15, GCr15SiMn | 高硬度，高耐磨性 |
| 弹簧 | 60Si2Mn, 50CrVA | 高弹性极限，疲劳强度好 |
| 模具 | Cr12, 5CrNiMo | 高硬度，耐磨，耐热 |
| 耐腐蚀件 | 304, 316不锈钢 | 优良耐腐蚀性 |
| 轻质结构件 | 6061, 7075铝合金 | 比强度高，重量轻 |
| 耐磨衬板 | 高锰钢 (ZGMn13) | 高韧性，加工硬化性好 |

### 热处理对性能的影响

| 热处理工艺 | 适用材料 | 效果 | 应用 |
|------------|----------|------|------|
| 退火 | 各种钢 | 降低硬度，改善切削性 | 预备热处理 |
| 正火 | 低碳钢、中碳钢 | 细化晶粒，均匀组织 | 预备热处理 |
| 淬火+回火 | 中碳钢、合金钢 | 提高强度、硬度 | 最终热处理 |
| 表面淬火 | 中碳钢 | 表面硬，心部韧 | 齿轮、轴类 |
| 渗碳 | 低碳钢 | 表面高碳，心部低碳 | 齿轮、凸轮 |
| 氮化 | 合金钢 | 表面高硬度，耐磨 | 精密零件 |

### 材料状态代号

| 代号 | 含义 | 说明 |
|------|------|------|
| O | 退火状态 | 最软状态，用于深冲 |
| H | 加工硬化状态 | H1x: 应变硬化；H2x: 硬化+退火 |
| T | 热处理状态 | T4: 固溶处理；T6: 固溶处理+人工时效 |
| F | 自由加工状态 | 未经热处理，机械性能不确定 |
| W | 固溶处理状态 | 不稳定状态，会自然时效 |

## 总结

正确选择材料需要综合考虑使用要求、工艺性能和经济性。在实际设计中，建议：
1. 优先选用标准材料和常用牌号
2. 考虑材料的加工工艺性
3. 注意材料的供应情况和成本
4. 必要时进行材料试验验证
