# 机械设计手册知识库

本目录包含《机械设计手册》第六版的Markdown格式知识库，可用于CAE-CLI学习中心加载。

## 目录结构

```
机械设计手册-md/
├── index.md                    # 知识库索引文件
├── 01_第1篇_一般设计资料.md    # 第1篇
├── 02_第2篇_机械制图_极限与配合_形状和位置公差及表面结构.md  # 第2篇
├── 03_第3篇_常用机械工程材料.md # 第3篇
├── 04_第4篇_机构.md           # 第4篇
├── 05_第5篇_机械传动.md       # 第5篇
├── 06_第6篇_液压传动.md       # 第6篇
├── 07_第7篇_气压传动.md       # 第7篇
├── 08_第8篇_液压控制.md       # 第8篇
├── 09_第9篇_电气控制.md       # 第9篇
└── images/                    # 图片目录（引用原图）
```

## 章节说明

| 序号 | 篇名 | 简介 |
|------|------|------|
| 1 | 一般设计资料 | 数学公式、物理常数、计量单位、优先数系 |
| 2 | 机械制图、极限与配合、形状和位置公差及表面结构 | 制图标准、公差配合、表面质量 |
| 3 | 常用机械工程材料 | 金属材料、非金属材料、复合材料 |
| 4 | 机构 | 平面机构、连杆机构、凸轮机构、齿轮机构 |
| 5 | 机械传动 | 螺旋传动、带传动、链传动、齿轮传动 |
| 6 | 液压传动 | 液压原理、液压元件、液压系统 |
| 7 | 气压传动 | 气压原理、气动元件、气动系统 |
| 8 | 液压控制 | 液压控制原理、控制系统 |
| 9 | 电气控制 | 低压电器、PLC控制、变频器 |

## 图片引用

图片引用保持原有格式，可引用原文件夹中的图片：

```markdown
![图名](../机械设计手册/images/图片文件名.jpg)
```

原图片位于：`knowledge/机械设计手册/images/`

## 使用说明

1. **搜索功能**：使用关键词搜索相关章节内容
2. **快速导航**：通过index.md索引表快速定位需要的篇章
3. **知识关联**：相关篇章之间通过关键词相互关联
4. **学习中心加载**：将本目录配置为知识库路径即可被学习中心加载
