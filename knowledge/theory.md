# 理论力学

理论力学研究刚体的平衡和运动规律。

## 静力学基础

### 1. 力的基本性质

- 力的三要素：大小、方向、作用点
- 力的合成与分解
- 力的可传性

### 2. 平衡条件

```
ΣFx = 0
ΣFy = 0
ΣM = 0
```

### 3. 约束与约束力

- 固定铰链
- 活动铰链
- 固定端
- 链杆

## 动力学基础

### 1. 牛顿三定律

- 第一定律（惯性定律）
- 第二定律（F=ma）
- 第三定律（作用反作用）

### 2. 动能定理

```
W = ΔEk
```

### 3. 动量定理

```
I = Δp
```

## 常用公式

### 直线运动

```
v = v₀ + at
s = v₀t + ½at²
v² = v₀² + 2as
```

### 曲线运动

```
v = ωr
a = αr
F = mv²/r
```

### 转动惯量

常见几何体转动惯量：
- 实心球：I = ²/₅mr²
- 实心圆柱：I = ½mr²
- 细杆：I = ¹/₃ml²
